## Getting Started
First, install:
```bash
npm i
npm i @chakra-ui/react @emotion/react @emotion/styled framer-motion
```
Then run
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
